# IPTVLiveChecker
IPTV直播源检测工具 Beta 1.0
开发者：半步沧桑
反馈邮箱：xiaomiren0510@gmail.com

## 功能清单
- 批量检测 IPTV 直播源可用性
- 支持 ffprobe / ffmpeg / MediaInfo 媒体解析
- 自动识别视频分辨率、音视频编码格式
- 链接解析、搜索、频道分组管理
- 直播源批量生成、合并、导出M3U/TXT
- IP归属地查询、网络响应速度测速
