# 2026-07-23 工作日志

## 项目分析：IPTVLiveChecker（IPTV 直播源检测工具 v1.0-beta）
- 技术栈：.NET Framework 4.7.2 WinForms，x64 为主；引用 WebView2（反射延迟绑定，避免硬依赖）、System.Web.Extensions/JavaScriptSerializer。
- 规模：C# 总计约 21.9k 行，其中 `IPTVLiveCheckerMain.cs` 单文件 20,941 行（巨石架构，所有 UI + 业务逻辑 + 自定义控件 + 图标生成 + 主题都在一个 Form 类里）。
- 子项目：`GenerateEncryptedMd5`（Release 构建后生成 AES 加密 MD5 写入 App.config 防篡改）、`Updater`（下载 zip + MD5 校验 + 备份 + 覆盖）。
- 功能：批量检测/解析（M3U、Tvbox、zhgx、kutv、huashi 多格式）、ffprobe/ffmpeg/MediaInfo 媒体解析（分辨率/编码）、IP 归属地查询（ip-api/ipwho.is/ping0.cc）、测速、M3U/TXT 导出、自动下载 FFmpeg/MediaInfo、自动更新。
- 安全机制：运行时校验自身 EXE MD5（AES 密钥材料分散写在代码里，仅"提高反编译难度"，非真正安全）；更新 MD5 校验 + 备份。
- 注意点：update.json 当前 isForceUpdate=true 但 versionCode==本地(100)，不会触发；一旦发布更高 versionCode 即强制更新（beta 偏激进）。下载依赖外部站点（github/gyan.dev），受限网络不可用。
- 主要风险：巨石单文件→维护/编译/合并困难、无单元测试/CI、网络外部依赖、async void 事件需关注 UI 卡顿。
